# Compiler_Pascal

GCC Compile Command:

gcc -o result CODE.C MAIN.C SCAN.C ANALYZE.C CGEN.C PARSE.C SYMTAB.C UTIL.C 

In Visual Studio, you should delete the TM.C file and compile the project.

------
C equivalents of Pascal types:
			Integer → long int/
			Char → unsigned char/
			Boolean → char/
			Real → double/
			Single → float/
